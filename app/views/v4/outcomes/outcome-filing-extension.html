Requesting more time to file

Before you contact us

If you need to request more time to file accounts before the deadline has passed, use our Apply to extend your accounts filing deadline service.

There is also guidance for applying for more time to file your company’s accounts.

If the deadline for filing the confirmation statement or company accounts has passed and you intend to file them, contact us using this online form instead.

If you still need help

You can contact us in one of the following ways:

Live chat

We'll connect you to one of our live chat customer service advisors. Live chat is available between 8:30am and 6pm, Monday to Friday.

Email

You can fill in the contact form to ask your query.

We'll respond by email, usually within 2 working days.

Phone

You'll need your reference code if you call. This code helps us direct you to the right support team.

Post

You can also write to Companies House, but you’ll usually get a faster response by using live chat, sending an online form or phoning us.
